# sprachkurs
An interactive web app with three games to learn the german language. Users solve drag-and-drop and matching tasks with instant feedback for self-guided practice.

Projektübersicht
Dieses Repository enthält drei browserbasierte Anwendungen:
Antonyme-Trainer: Ein Kachel-basiertes Quiz zum Einprägen von Gegenteilen.
Präpositionen-Zuordnung: Ein Drag-and-Drop-Spiel zur visuellen Grammatikschulung.
Deklinations-Quiz: Ein randomisiertes Quiz zur korrekten Anwendung von Nomina.


Architektur & Technologien
HTML5 & CSS3: Struktur und Design
JavaScript: Dynamische Inhaltsgenerierung, DOM-Manipulation und Event-Handling.
jQuery: Effiziente Validierung von Benutzereingaben im Präpositionen-Modul.
FontAwesome: Vektorgrafiken für eine konsistente Ikonografie.


Installation & Nutzung
Da es sich um eine reine Frontend-Anwendung handelt, ist keine serverseitige Installation notwendig:
1. Repository klonen
2. Die Datei index.html im Browser öffnen
